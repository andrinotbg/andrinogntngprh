Metode Menampilkan Data di JavaScript

  1. Menggunakan console.log()
    Metode ini menampilkan data di konsol pengembang (developer console).
  2. Menggunakan alert()
    Metode ini menampilkan data dalam kotak dialog (alert box).
  3. Menggunakan document.write()
    document.write() adalah metode yang digunakan untuk menulis konten langsung ke dalam dokumen HTML. Namun, penggunaannya sudah tidak disarankan dalam pengembangan web modern karena dapat mengganggu struktur halaman jika digunakan setelah halaman sepenuhnya dimuat.

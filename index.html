Kelebihan HTML5

HTML5 memiliki sejumlah kelebihan dibandingkan dengan HTML:

  • Elemen semantik seperti <header>, <nav>, dan lainnya.
  • Elemen <audio> dan <video> memungkinkan penyisipan multimedia.
  • LocalStorage untuk penyimpanan data lokal.
  • Elemen <canvas> untuk visualisasi data.

CSS

  • Pengaturan tampilan dan tata letak elemen pada halaman web.
  • Desain responsif untuk berbagai ukuran layar.

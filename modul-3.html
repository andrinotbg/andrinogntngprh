Pengenalan HTML dan CSS

HTML (HyperText Markup Language)

HTML adalah bahasa markup yang digunakan untuk membuat struktur halaman web. Elemen HTML dibentuk oleh tag, yang biasanya terdiri dari tag pembuka dan tag penutup. Beberapa komponen utama HTML meliputi:

  • Elemen: Bagian dasar dari HTML, seperti <h1> untuk judul, <p> untuk paragraf, dan <div> untuk pembagian konten.
  • Atribut: Menambahkan informasi tambahan ke elemen, seperti src pada tag <img> untuk menentukan sumber gambar.
  • Struktur: HTML membentuk struktur hierarkis dengan penggunaan tag untuk membuat bagian-bagian berbeda dalam sebuah halaman (misalnya, header, footer, dan konten utama).

CSS (Cascading Style Sheets)

CSS adalah bahasa yang digunakan untuk mengatur tampilan dan format elemen yang dibuat dengan HTML. Dengan CSS, kamu bisa mengubah warna, font, layout, dan banyak aspek visual lainnya dari halaman web. Beberapa konsep utama CSS meliputi:

  • Selector: Menentukan elemen mana yang akan dipengaruhi oleh aturan CSS. Contohnya, p untuk memilih semua paragraf.
  • Properties dan Values: Mengatur gaya dengan mendefinisikan property (misalnya, color, font-size, margin) dan memberikan nilai untuk property tersebut (misalnya, red, 16px, 10px).
  • Box Model: Konsep yang menjelaskan bagaimana elemen HTML memiliki margin, border, padding, dan konten.
